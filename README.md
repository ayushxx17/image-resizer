#  Image Resizer Tool

Resize and convert all images in a folder using Python and Pillow.

##  Features
- Resize to 400x400 px
- Converts all images to PNG
- Works in batch mode

##  Folders
- `input_images/` → original JPG/PNG images
- `output_images/` → resized PNGs

##  Requirements
- Python
- Pillow (`pip install Pillow`)
